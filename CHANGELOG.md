# 📜 Lịch sử thay đổi (Changelog)

Tất cả các thay đổi đáng chú ý đối với dự án **FoodBee** sẽ được ghi lại trong file này.

## [1.0.0] - 2026-04-26
### 🚀 Tính năng mới
- **Hệ sinh thái đa nền tảng:** Khởi tạo cấu trúc dự án cho Web (React), Mobile (React Native) và Backend (Laravel 11).
- **AI Chatbot:** Tích hợp module gợi ý món ăn thông minh sử dụng Python.
- **Thanh toán tự động:** Tích hợp PayOS cho quy trình thanh toán QR nhanh chóng.
- **Real-time:** Hỗ trợ thông báo và cập nhật trạng thái đơn hàng thời gian thực qua WebSockets.

### 🛠️ Quản lý dự án
- Thiết lập bộ mẫu **GitHub Issue & Pull Request**.
- Cập nhật tài liệu hướng dẫn đóng góp (**CONTRIBUTING.md**) và quy tắc ứng xử (**CODE_OF_CONDUCT.md**).
- Công bố dự án dưới giấy phép mã nguồn mở **GPL-3.0**.
- Tối ưu hóa giao diện README với sơ đồ cấu trúc và hình ảnh minh họa sắc nét.
