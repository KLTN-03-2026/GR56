# 🤝 Hướng dẫn đóng góp vào FoodBee

Cảm ơn bạn đã quan tâm đến việc đóng góp cho dự án FoodBee! Chúng tôi rất trân trọng mọi sự giúp đỡ từ cộng đồng.

## 🚀 Quy trình phát triển (Workflow)

Để đảm bảo lịch sử Git sạch sẽ và dễ quản lý, chúng tôi khuyến khích sử dụng công cụ hỗ trợ có sẵn:

1.  **Fork** dự án về tài khoản của bạn.
2.  **Clone** dự án về máy local.
3.  Sử dụng script **`./git-rules/git-menu.sh`** để quản lý các thay đổi:
    -   Script này giúp bạn tự động thêm tiền tố (feat, fix, docs...) theo chuẩn Conventional Commits.
    -   Hỗ trợ đẩy code nhanh lên đúng nhánh đang làm việc.

## 📜 Quy chuẩn Commit Message

Chúng tôi tuân thủ chuẩn **Conventional Commits**:
-   `feat:` Một tính năng mới.
-   `fix:` Sửa một lỗi kỹ thuật.
-   `docs:` Thay đổi về tài liệu.
-   `style:` Thay đổi không ảnh hưởng đến logic code (white-space, formatting...).
-   `refactor:` Thay đổi code nhưng không sửa lỗi cũng không thêm tính năng.
-   `perf:` Thay đổi code để cải thiện hiệu năng.
-   `test:` Thêm hoặc sửa các bài kiểm tra.

## 🛠️ Yêu cầu kỹ thuật
-   **Backend:** PHP 8.2+, Laravel 11.
-   **Frontend:** Node.js 18+, React 19.
-   **Mobile:** React Native, Expo.
